# Contracts: Shared UI Components

This folder defines the **public contracts** for the shared UI component layer. These contracts are the interface between the shared components (`src/components/ui/`) and the consuming feature screens.

## Files

| Contract | Purpose |
|----------|---------|
| `component-api-contract.md` | Public API surface for all shared components (props, subparts, naming) |
| `figma-component-map.md` | Mapping between Figma nodes and component implementations |
| `a11y-baseline.md` | Accessibility requirements every shared component must meet |
